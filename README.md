## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/People/Student.png" alt="Student" width="35" height="35" /> Introduction

This is a modern, scalable, and secure backend application for managing student records efficiently. Built using the **Java Spring Boot** framework, this system provides a robust REST API to handle all necessary CRUD (Create, Read, Update, Delete) operations regarding student information.

### <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Objects/Light%20Bulb.png" alt="Key Features" width="25" height="25" /> Key Features
* **Full CRUD:** Managed via standard REST endpoints.
* **Persistence:** Data stored securely in MySQL database.
* **Validation:** Severe input validation using Hibernate Validator.
* **Documentation:** Integrated with Swagger/OpenAPI UI for easy testing.
* **Except Handling:** Centralized global exception handling for clean API responses.

Make sure you have **Java 17+**, **Maven**, and **MySQL** installed.

Create a database named `sms_db` in MySQL. Update `src/main/resources/application.properties` with your credentials:
```properties

spring.datasource.username=your_username
